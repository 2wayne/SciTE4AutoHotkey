SciTE4AutoHotkey
================

Introduction
------------

SciTE4AutoHotkey is a SciTE-based AutoHotkey script editor. It offers:

* Syntax highlighting
* Code folding
* Calltips (also known as IntelliSense)
* AutoComplete
* AutoIndent
* AutoHotkey help integration
* Abbreviations
* AutoHotkey_L debugging support
* Tools for AutoHotkey scripting
* A toolbar that enables easy access to the tools
* Some AutoHotkey scripting facilities

Git importing status
--------------------

The code is complete, but there are no instructions yet on how to build it. The recommended "setup" procedure is fetching the repository somewhere, then making a symbolic link between %AhkDir%\SciTE and <repo-dir>\source. \*.exe, \*.dll, \*.chm and \*.bak are already discarded by .gitignore.
